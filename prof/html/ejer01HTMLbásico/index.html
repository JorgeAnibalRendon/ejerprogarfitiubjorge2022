Preguntas
Incluir respuestas den algún punto del ejercicio práctico elegido por ustedes.

  • 1) Dar un par de ejemplos de uso de etiquetas meta.

    Primer ejemplo: Se puede usar para especificar el tipo de codeado de caracteres que se desea utilizar. Ejemplo:
    meta charset="UTF-8".

    También se puede usar para que haga determinadas acciones. Por ejemplo, con la siguiente línea de código, la página se redirige después de 5 segundos:
    meta http-equiv="refresh" content="5;url=https://www.mozilla.org".

  • 2) Nombrar al menos 2 atributos aplicables a la etiqueta HTML

    version: especifica la versión Document Type Definition de HTML del documento actual.
    xmlns: especifica el espacio de nombres XML del documento.

  • 3) ¿Cómo se relaciona un header http con la etiqueta head de HTML?

  • 4) ¿Qué es UTF-8?

    Es la codificación de caracteres más común de la World Wide Web. Cada carácter está representado por uno a cuatro bytes.

  • 5) ¿Qué atributo HTML sirve para aplicar estilo? ¿A qué etiqueta es aplicable?

    Se utiliza el atributo "meta" para aplicar esto y es aplicable a la etiqueta "head".

  • 6) ¿Dónde se indica que un enlace debe ubicar el documento en una nueva pestaña?

    Cuando se hace un a:href se especifica esa función agregando target="_blank"

  • 7) Explicar cuando usarían etiquetas semánticas y con que propósito

    Dependiendo la situación, las utilizaría para, por ejemplo, enfatizar un titulo (usando "b"=>bold), dar estilo a un texto o a un nombre (usando "i"=>Italic o "em"=>Emphasized)

  • 8) ¿Qué atributos de estilo básicos aplicarían a una tabla? Explicar

    Utilizaría espaciamientos, como "margin" o "padding" para acomodar mejor las tablas. Y para los colores utilizaría "background-color"

  • 9) ¿Qué diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

    El elemento body en el documento HTML contiene todo el contenido de un documento HTML, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc. Mientras que el body en el protocolo HTTP son los bytes de datos transmitidos en un mensaje de transacción HTTP inmediatamente después de los encabezados, si hay alguno.

  • 10) ¿Qué atributos marcan la agrupación de celdas de una tabla? ¿A qué etiquetas se aplican?

    Se le aplican a la etiqueta "cellpadding", la cual sirve para poder ajustar el tamaño de la separación de las celdas
